Object Factory

Understand Object-Oriented Programming (OOP). The Class is the blueprint, and Objects are the actual instances created from it.

Class Definition (Blueprint)
class Robot: def __init__(self, name, color): self.name = name self.color = color self.battery = 100 def introduce(self): print(f"I am {self.name}") def use_power(self): self.battery -= 20
